Teaching Aid Generator
======================

This repository now uses `README.md` as the primary project guide for GitHub.

Quick start
-----------

1. Install dependencies:
   `pip install -r requirements.txt`
2. Add your API keys to `.env`
3. Run the Streamlit studio:
   `streamlit run app.py`

Files
-----

- `README.md` - full setup and usage guide
- `app.py` - Streamlit Studio UI
- `run.py` - CLI workflow
- `generate_content.py` - Groq content generation
- `generate_pdf.py` - Markdown to PDF conversion
- `image_gen.py` - image directive handling and Google image generation
